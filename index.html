리베로 (Libero)

아마추어 9인제 배구 동호회 캡틴을 위한 Android 앱

소개

리베로는 한국 아마추어 9인제 배구 동호회 주장(캡틴)과 총무가 토너먼트 일정, 회비 관리, 출석 확인을 한곳에서 처리할 수 있도록 돕는 Android 앱입니다.

주요 기능

  • 토너먼트 알림: Daum 카페에 새로운 토너먼트 공고가 올라오면 자동으로 푸시 알림을 보냅니다.
  • 회비 대사: KakaoBank 엑셀 명세서를 업로드하면 입금 내역을 자동으로 매칭해 줍니다.
  • 밴드 자동 게시: 동호회 밴드에 주간 출석 체크 게시물을 원하는 시간에 자동으로 올려 줍니다.

Band Open API 사용

리베로는 캡틴이 본인이 운영하는 동호회 밴드에 주간 출석 체크 게시물을 자동으로 올릴 수 있도록 Band Open API의 글 작성 권한(WRITE_POST)을 사용합니다. 사용자는 OAuth 동의 화면에서 본인의 밴드를 선택하고 권한을 명시적으로 허용한 경우에만 글이 게시됩니다.

현재 상태

포트폴리오 프로젝트로 개발 중이며, Google Play Console 비공개 테스트를 준비하고 있습니다. MVP 기준 예상 사용자 50-100명, 일일 자동 게시 수 100건 이하로 제한됩니다.
